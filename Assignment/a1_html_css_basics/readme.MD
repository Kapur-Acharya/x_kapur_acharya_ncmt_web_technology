# A1_HTML_CSS_BASICS

## Overview
A responsive webpage built with HTML5 and CSS3 as part of Assignment 1.

## Files
```
A1_HTML_CSS_BASICS/
├── index.html
├── style.css
└── images/
    └── image.jpg
```

## What's Included

**HTML**
- Semantic elements: `<header>`, `<main>`, `<article>`, `<figure>`, `<figcaption>`
- Image with meaningful alt text

**Form Inputs**
- Email, Date, Range, URL, Number, Color
- `required` validation on key fields
- Regex pattern for 9-digit number: `^[0-9]{9}$`

**CSS**
- Clean typography, spacing, and color scheme
- Mobile-first responsive design
- Media query for larger screens
- CSS animation on a page element

## How to Run
Open `index.html` in any modern web browser. No setup or installation required.

## Tested On
- Chrome / Firefox / Edge
- Mobile and desktop screen sizes
